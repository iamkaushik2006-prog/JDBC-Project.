# JDBC Project (Banking + Hotel System)

This project is made using Java and JDBC to understand how Java connects with a database.
It contains two small systems – a banking system and a hotel reservation system.

---

## Banking System

In this part, I tried to implement basic banking features like:

* Register and Login user
* Create account
* Debit and Credit money
* Transfer money between accounts
* Check account balance

---

## Hotel Reservation System

This module handles simple hotel operations:

* Book a room
* View reservations
* Get room number using reservation id
* Update reservation details
* Delete reservation

---

## Technologies Used

* Java
* JDBC
* MySQL

---

## How to Run

1. Add MySQL connector jar to the project
2. Create databases:

    * banking_system
    * hotel_db
3. Update username and password in code
4. Run the Java files

---

## Purpose of this Project

I made this project to practice:

* JDBC connectivity
* SQL queries in Java
* Handling real-world scenarios like transactions
